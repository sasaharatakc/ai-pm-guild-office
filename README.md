# AI PM OS

AI PM OS is an Enterprise Intelligence Operating System for operating an AI company where humans and AI employees work together through missions, planning, execution, knowledge, governance, and experience.

## Current phase

Blueprint v1.0 is treated as frozen. This repository is now the reference monorepo for implementation.

## Repository goals

- Build the AI PM OS reference monorepo
- Generate contracts, schemas, SDKs, and tests from canonical specifications
- Implement the Execution Cluster MVP
- Build the AI Company / AI Employee Office UI
- Manage work through 1000-level issue decomposition

## Core product model

Organization → Mission → Planning → Execution → Employee → Capability → Execution Cluster → Knowledge → Business Value
